🔧 Galeri Sorun Giderme Kontrol Listesi

1. Görsel Yükleme Kontrolü

Browser DevTools Console'unda aşağıdaki kodu çalıştırın:

// Resimler yüklendi mi kontrol et const images = document.querySelectorAll('.gallery-item img'); images.forEach((img, i) => { const hasSrc = img.src && !img.src.includes('data:image/svg'); const hasDataSrc = img.dataset.src; console.log(`Resim ${i}:`, { placeholder: img.src.substring(0, 30) + '...', dataSrc: hasDataSrc ? 'VAR ✓' : 'YOK ✗', loaded: hasSrc ? 'YÜKLENDI ✓' : 'BEKLENIYOR ⏳' }); });

Beklenen sonuç: Tüm resimlerin dataSrc: VAR olmalı ve visible olanlar loaded: YÜKLENDI olmalı.

2. Lazy Loading Başladı mı?

Console'da yazın:

// Lazy loader nesnesi var mı kontrol et if (window.GalleryLazyLoader) { console.log('✓ GalleryLazyLoader sınıfı yüklendi'); console.log('✓ Lazy loading aktif'); } else { console.log('✗ GalleryLazyLoader bulunamadı - lazy-loading.js yüklü mü?'); }

Beklenen sonuç: ✓ Yüklendi mesajı görülmeli.

3. IntersectionObserver Desteği

Console'da yazın:

// Browser'ın IntersectionObserver desteği console.log( 'IntersectionObserver destekleniyor mu?', 'IntersectionObserver' in window ? '✓ EVET' : '✗ HAYIR' );

Beklenen sonuç: ✓ EVET çıkmalı (modern tarayıcılarda).

4. Mobilde Lightbox Tam Ekran Testi

Adımlar:

  1. Chrome DevTools'ta Device Mode'a geçin (F12 → Ctrl+Shift+M)
  2. Herhangi bir galeri resmine tıklayın
  3. Resmin tam ekranda, sapıtmadan gösterilip gösterilmediğini kontrol edin
  4. Kapatma butonuna (×) tıklayın - kapanmalı
  5. Ok tuşları (← →) ile resim geçişi çalışmalı

Beklenen sonuç: Tüm işlemler sorunsuz çalışmalı.

5. Network Sekmesi Kontrolü

Adımlar:

  1. DevTools → Network sekmesi
  2. Sayfayı yenile (F5)
  3. Kaç resim indirileceğini sayın (başlangıçta 6-8 olmalı)
  4. Sayfayı aşağı kaydırın
  5. Yeni resimler indiriliyor mu kontrol edin (evet olmalı)

Beklenen sonuç: Başlangıçta az resim, sonra gerekli olduğunda yüklenme.

6. Console Hataları Kontrolü

Adımlar:

  1. DevTools → Console sekmesi
  2. Red X işareti (hata) var mı kontrol edin
  3. Yellow (uyarı) var mı kontrol edin

Beklenen sonuç: Hiç hata olmamalı. Uyarılar normal.
